5. Create `src/persistence/rdb.rs` with custom binary format:
   - Constants: `RDB_MAGIC = b"RUSTREDIS"`, `RDB_VERSION = 1u8`, type tags (STRING=0, HASH=1, LIST=2, SET=3, SORTED_SET=4), `EOF_MARKER = 0xFF`, `DB_SELECTOR = 0xFE`
   - `pub fn save(databases: &[Database], path: &Path) -> anyhow::Result<()>`:
     - Iterate databases, skip empty ones
     - For each non-empty db: write DB_SELECTOR byte + db index u8
     - For each (key, entry): skip if expired, write type tag, key (u32 len + bytes), TTL (i64: 0 for none, unix millis for future expiry using `SystemTime::now() + (expires_at - Instant::now())`), value data per type
     - String: u32 len + bytes
     - Hash: u32 field_count + (u32 field_len + field + u32 val_len + val) repeated
     - List: u32 elem_count + (u32 elem_len + elem) repeated
     - Set: u32 member_count + (u32 member_len + member) repeated
     - SortedSet: u32 member_count + (u32 member_len + member + f64 score) repeated
     - Footer: EOF_MARKER byte + CRC32 u32 of all preceding bytes
     - Write to `.tmp` file first, then `std::fs::rename` for atomicity
   - `pub fn load(databases: &mut [Database], path: &Path) -> anyhow::Result<usize>`:
     - Read file, verify magic + version
     - Parse entries: for each DB_SELECTOR read db index, then entries
     - Convert unix millis TTL back to Instant: `Instant::now() + Duration::from_millis((unix_ms - now_ms) as u64)` -- skip entries where TTL is in the past
     - Verify CRC32 checksum (all bytes except last 4 vs last 4 bytes as u32)
     - Return total keys loaded
     - On any error: log with tracing::error, return Err (caller decides to start empty)

6. Add `pub fn data_mut(&mut self) -> &mut HashMap<Bytes, Entry>` to Database in `src/storage/db.rs` (needed for RDB load to insert directly without going through set() which sets created_at).
