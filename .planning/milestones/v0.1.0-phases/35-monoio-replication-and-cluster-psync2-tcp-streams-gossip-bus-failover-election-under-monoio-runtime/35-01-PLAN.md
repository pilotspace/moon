Monoio read pattern:
```rust
use monoio::io::{AsyncReadRent, AsyncWriteRentExt};
// Ownership read: takes buf, returns (Result, buf)
let tmp_buf = vec![0u8; 8192];
let (result, buf) = stream.read(tmp_buf).await;
let n = result?;
// Copy into BytesMut: read_buf.extend_from_slice(&buf[..n]);
```

Monoio write pattern:
```rust
// Ownership write: takes Vec<u8>, returns (Result<usize>, Vec<u8>)
let data = some_bytes.to_vec();
let (wr, _) = stream.write_all(data).await;
wr?;
```

Monoio spawn/select/sleep:
```rust
monoio::spawn(async move { ... });
monoio::select! { ... }
monoio::time::sleep(Duration::from_millis(10)).await;
```

Key difference from tokio:
- No OwnedWriteHalf in monoio — use full TcpStream with Arc<parking_lot::Mutex<>> for shared write access
- No read_exact — must loop on read() accumulating into buffer until enough bytes
- No read_buf(&mut BytesMut) — read(Vec<u8>) takes ownership, copy result into BytesMut
- monoio::net::TcpStream::connect takes &SocketAddr not &str

**read_line (monoio):**
- Cannot use `read_exact(&mut [0u8; 1])` — monoio has no read_exact for borrowed buffers.
- Use ownership read with 1-byte buffer in a loop:
  ```rust
  let mut line = Vec::new();
  loop {
      let buf = vec![0u8; 1];
      let (result, buf) = stream.read(buf).await;
      let n = result?;
      if n == 0 { anyhow::bail!("EOF during read_line"); }
      let byte = buf[0];
      if byte == b'\n' && line.last() == Some(&b'\r') {
          line.pop();
          return Ok(line);
      }
      line.push(byte);
      if line.len() > 4096 { anyhow::bail!("Line too long"); }
  }
  ```
  NOTE: Reading 1 byte at a time is acceptable here because read_line is only called during the PSYNC handshake (5 times total), not in the hot path. For efficiency, could read larger chunks and buffer, but handshake simplicity is preferred.

**read_bulk_string (monoio):**
- Call monoio `read_line` to get length.
- Then read `len + 2` bytes using a loop with ownership read:
  ```rust
  let mut data = vec![0u8; len + 2];
  let mut offset = 0;
  while offset < data.len() {
      let remaining = data[offset..].to_vec();
      let (result, buf) = stream.read(remaining).await;
      let n = result?;
      if n == 0 { anyhow::bail!("EOF during read_bulk_string"); }
      data[offset..offset + n].copy_from_slice(&buf[..n]);
      offset += n;
  }
  data.truncate(len);
  ```
