   - Add `pub fn unlink(db: &mut Database, args: &[Frame]) -> Frame`:
     - If args is empty, return err_wrong_args("UNLINK").
     - For each arg, extract key, call `db.remove(key)`. If Some(entry), increment count.
     - For large values (Hash len > 64, List len > 64, Set len > 64, SortedSet members len > 64), call `tokio::task::spawn_blocking(move || drop(entry))` to defer the drop.

     - Parse optional MATCH pattern, COUNT hint (default 10), TYPE filter from remaining args (pairs like "MATCH" "pattern", "COUNT" "10", "TYPE" "string").

     - Starting from cursor position, iterate up to COUNT keys, applying MATCH (use glob_match from this module) and TYPE post-filter.
     - Return Frame::Array with 2 elements: [Frame::BulkString(next_cursor_as_string), Frame::Array(matched_keys)]. Return cursor "0" when done.
