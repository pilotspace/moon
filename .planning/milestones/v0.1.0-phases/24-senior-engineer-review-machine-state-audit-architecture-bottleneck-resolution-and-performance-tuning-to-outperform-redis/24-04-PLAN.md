**Step 1: Build optimized release binary.**
```bash
cargo build --release
```

**Step 2: Benchmark Redis 7.x baseline.**
Start Redis server on port 6379 (if not already running):
```bash
redis-server --port 6379 --daemonize yes --save ""
```

Run redis-benchmark against Redis:
```bash
redis-benchmark -h 127.0.0.1 -p 6379 -c 10 -n 100000 -t get,set -d 256 -q
redis-benchmark -h 127.0.0.1 -p 6379 -c 50 -n 100000 -t get,set -d 256 -q
redis-benchmark -h 127.0.0.1 -p 6379 -c 100 -n 100000 -t get,set -d 256 -q
```
Capture ops/sec for SET and GET at each concurrency level.

**Step 3: Benchmark rust-redis.**
Start rust-redis on port 6380:
```bash
./target/release/rust-redis --port 6380 &
RREDIS_PID=$!
sleep 1
```

Run same benchmarks:
```bash
redis-benchmark -h 127.0.0.1 -p 6380 -c 10 -n 100000 -t get,set -d 256 -q
redis-benchmark -h 127.0.0.1 -p 6380 -c 50 -n 100000 -t get,set -d 256 -q
redis-benchmark -h 127.0.0.1 -p 6380 -c 100 -n 100000 -t get,set -d 256 -q
```

Kill the servers:
```bash
kill $RREDIS_PID 2>/dev/null
redis-cli -p 6379 SHUTDOWN NOSAVE 2>/dev/null
```

If redis-benchmark or redis-server are not available, document that benchmarks need to be run manually.
