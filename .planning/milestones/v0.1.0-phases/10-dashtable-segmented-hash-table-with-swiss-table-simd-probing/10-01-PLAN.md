**2. Create `src/storage/dashtable/simd.rs` -- SIMD group operations:**

```rust
/// A group of 16 control bytes for Swiss Table SIMD probing.
#[repr(C, align(16))]
pub struct Group(pub [u8; 16]);

pub const EMPTY: u8 = 0xFF;
pub const DELETED: u8 = 0x80;

/// Bitmask of matching positions (implements Iterator yielding set bit indices).
pub struct BitMask(pub u16);
```

Implement on `Group`:
- `match_h2(&self, h2: u8) -> BitMask` -- SSE2 path: `_mm_load_si128` + `_mm_set1_epi8(h2 as i8)` + `_mm_cmpeq_epi8` + `_mm_movemask_epi8`. Use `#[cfg(target_arch = "x86_64")]` with `#[target_feature(enable = "sse2")]`. Scalar fallback for aarch64/other: byte-by-byte loop building mask.
- `match_empty(&self) -> BitMask` -- calls `match_h2(EMPTY)`
- `match_empty_or_deleted(&self) -> BitMask` -- SSE2: `_mm_movemask_epi8(ctrl)` (top bit set means EMPTY or DELETED). Scalar: check `byte & 0x80 != 0`.

Implement on `BitMask`:
- `any_set(&self) -> bool`
- `lowest_set_bit(&self) -> Option<usize>` -- `trailing_zeros()`
- `Iterator` impl: yield each set bit position, clear lowest with `self.0 &= self.0 - 1`

**3. Create `src/storage/dashtable/segment.rs` -- Segment with 60 slots:**

```rust
use std::mem::MaybeUninit;

/// Total slots per segment: 56 regular + 4 stash = 60
pub const REGULAR_SLOTS: usize = 56;
pub const STASH_SLOTS: usize = 4;
pub const TOTAL_SLOTS: usize = REGULAR_SLOTS + STASH_SLOTS; // 60
/// Load threshold: 85% of 60 = 51 slots
pub const LOAD_THRESHOLD: usize = 51;

/// Control byte groups: 4 groups of 16 bytes = 64 bytes
/// Groups 0-3 cover slots 0-15, 16-31, 32-47, 48-63
/// Slots 56-59 are stash (within group 3, positions 8-11)
/// Slots 60-63 are padding (always EMPTY)
#[repr(C)]
pub struct Segment<K, V> {
    ctrl: [Group; 4],           // 64 bytes, 16-aligned due to Group alignment
    keys: [MaybeUninit<K>; TOTAL_SLOTS],
    values: [MaybeUninit<V>; TOTAL_SLOTS],
    count: u32,
    depth: u32,
}
```

CRITICAL SAFETY: Implement `Drop` for `Segment` that iterates ctrl bytes 0..TOTAL_SLOTS, and for each slot where `ctrl_byte != EMPTY && ctrl_byte != DELETED`, calls `assume_init_drop()` on both `keys[i]` and `values[i]`.

Implement helper `fn h2(hash: u64) -> u8` returning `(hash >> 57) as u8 & 0x7F` (top 7 bits, ensure MSB is 0 so it's distinguishable from EMPTY/DELETED).

Implement on `Segment`:
- `new(depth: u32) -> Self` -- all ctrl bytes set to EMPTY, count=0
- `ctrl_byte(&self, slot: usize) -> u8` -- read control byte at slot index (group = slot/16, pos = slot%16)
- `set_ctrl_byte(&mut self, slot: usize, byte: u8)` -- write control byte
- `is_full(&self) -> bool` -- `count >= LOAD_THRESHOLD`
- `find<Q: ?Sized>(&self, h2: u8, key: &Q, bucket_a: usize, bucket_b: usize) -> Option<usize>` where K: Borrow<Q>, Q: Eq
  - For each of bucket_a, bucket_b: compute the group index (bucket / 16), call `group.match_h2(h2)`, for each match check actual key equality. Return slot index if found.
  - Then linear scan stash slots (56..60) checking ctrl byte == h2, then key equality.
  - Return None if not found.
- `get(&self, h2: u8, key: &Q, bucket_a: usize, bucket_b: usize) -> Option<&V>` -- calls find, returns value ref
- `get_mut(&mut self, h2: u8, key: &Q, bucket_a: usize, bucket_b: usize) -> Option<&mut V>` -- calls find logic inline (avoid borrow issues), returns mutable value ref
- `get_key_value(&self, h2: u8, key: &Q, bucket_a: usize, bucket_b: usize) -> Option<(&K, &V)>` -- needed for iteration/SCAN
- `insert(&mut self, h2: u8, key: K, value: V, bucket_a: usize, bucket_b: usize) -> InsertResult<V>` where InsertResult is `Inserted | Replaced(V) | NeedsSplit(K, V)`
  - First check if key exists (find). If so, replace value in-place, return Replaced(old_value).
  - If not found and segment not full: find first EMPTY or DELETED slot in bucket_a's group, then bucket_b's group, then stash. Write ctrl byte = h2, write key and value via MaybeUninit, increment count. Return Inserted.
  - If segment is full: return NeedsSplit(key, value) -- caller handles split.
- `remove<Q: ?Sized>(&mut self, h2: u8, key: &Q, bucket_a: usize, bucket_b: usize) -> Option<(K, V)>` where K: Borrow<Q>, Q: Eq
  - Find slot, read key+value via `assume_init_read()`, set ctrl to DELETED (or EMPTY if in stash), decrement count.
- `split(&mut self, hasher: &impl Fn(&K) -> u64) -> Segment<K, V>` -- per-segment split algorithm:
  - Create new segment with depth = self.depth + 1
  - For each occupied slot: rehash key, check bit at position `self.depth` in hash
  - If bit is 1: move entry (assume_init_read from old, insert into new), mark old slot EMPTY
  - If bit is 0: entry stays, but must be re-inserted into a fresh old segment layout (since slot positions may change with the new depth)
  - Actually simpler: collect all entries, clear old segment, re-insert entries with bit=0 into old, bit=1 into new
  - Increment self.depth

```rust
pub struct Iter<'a, K, V> {
    segments: &'a [Box<Segment<K, V>>],
    seg_idx: usize,
    slot_idx: usize,
    remaining: usize,
}

pub struct IterMut<'a, K, V> { /* same but &mut segments */ }
pub struct Keys<'a, K, V>(Iter<'a, K, V>);
pub struct Values<'a, K, V>(Iter<'a, K, V>);
```

Iteration logic: advance through segments, within each segment scan ctrl bytes for FULL slots (byte < 0x80), yield key-value references via `assume_init_ref()`. Skip EMPTY and DELETED. Track `remaining` count to short-circuit when all entries yielded.

Implement `Iterator` for all four types. `Iter` yields `(&K, &V)`, `IterMut` yields `(&K, &mut V)`, `Keys` yields `&K`, `Values` yields `&V`. Implement `size_hint` returning `(remaining, Some(remaining))`.

IMPORTANT: DashTable directory may have duplicate segment pointers (extendible hashing). The iterator must deduplicate -- track which segments have been visited. Since segments have unique addresses, use a simple seen-depth approach: iterate by unique segment index. The DashTable should provide a method `unique_segments()` that deduplicates directory entries by pointer address.

**2. Create `src/storage/dashtable/mod.rs` -- DashTable public API:**

```rust
pub mod simd;
pub mod segment;
pub mod iter;

use bytes::Bytes;
use std::hash::Hash;

pub struct DashTable<K, V> {
    directory: Vec<Box<Segment<K, V>>>,
    depth: u32,        // global depth (log2 of directory size)
    len: usize,        // total entry count across all segments
}
```

Hash routing functions:
```rust
fn hash_key(key: &[u8]) -> u64 {
    xxhash_rust::xxh64::xxh64(key, 0)
}

fn segment_index(hash: u64, depth: u32) -> usize {
    if depth == 0 { 0 } else { (hash >> (64 - depth)) as usize }
}

fn home_buckets(hash: u64) -> (usize, usize) {
    let a = ((hash >> 8) as usize) % REGULAR_SLOTS;
    let mut b = ((hash >> 16) as usize) % REGULAR_SLOTS;
    if b == a { b = (a + 1) % REGULAR_SLOTS; }
    (a, b)
}
```

Implement on `DashTable<Bytes, V>` (concrete key type since we hash &[u8]):
- `new() -> Self` -- 1 segment, depth=0
- `with_capacity(cap: usize) -> Self` -- pre-allocate segments for `cap` entries
- `len(&self) -> usize`
- `is_empty(&self) -> bool`
- `get(&self, key: &[u8]) -> Option<&V>` -- hash key, route to segment, call segment.get with h2 and home buckets
- `get_mut(&mut self, key: &[u8]) -> Option<&mut V>` -- same but mutable
- `contains_key(&self, key: &[u8]) -> bool` -- get().is_some()
- `insert(&mut self, key: Bytes, value: V) -> Option<V>` -- hash, route, segment.insert. If NeedsSplit: call `self.split_segment(seg_idx)`, then retry insert. Return None on new insert, Some(old) on replace.
- `remove(&mut self, key: &[u8]) -> Option<(Bytes, V)>` -- hash, route, segment.remove. Decrement len on success. Returns the key-value pair. Also provide `remove_entry` alias if needed.
  **CRITICAL:** The current `db.rs` calls `self.data.remove(key)` which on HashMap returns `Option<V>`. DashTable must match: `remove(&mut self, key: &[u8]) -> Option<V>` (drop key, return value only).
- `keys(&self) -> Keys<Bytes, V>`
- `values(&self) -> Values<Bytes, V>`
- `iter(&self) -> Iter<Bytes, V>`
- `iter_mut(&mut self) -> IterMut<Bytes, V>`

Private methods:
- `split_segment(&mut self, seg_idx: usize)` -- call segment.split(hash_fn), double directory if needed (when segment depth > global depth), update directory pointers
- `unique_segments(&self) -> Vec<(usize, &Box<Segment<K, V>>)>` -- deduplicate by pointer address for iteration

**Directory doubling logic:**
When a segment splits and its new depth exceeds the global depth:
1. Double the directory: for each entry at index i, copy pointer to new indices 2*i and 2*i+1
2. Increment global depth
3. Place old segment at the entry matching bit=0, new segment at entry matching bit=1
