  <action>
    Implement the full RESP2 streaming parser in `src/protocol/parse.rs` using the two-pass check-then-parse pattern from mini-redis:

    1. **Public API function** `pub fn parse(buf: &mut BytesMut, config: &ParseConfig) -> Result<Option<Frame>, ParseError>`:
       - Create a `Cursor::new(&buf[..])` for the check pass
       - Call `check(&mut cursor, config, 0)` -- third arg is current depth for recursion tracking
       - If check returns Ok(()), record `cursor.position() as usize` as `len`
       - Create a new cursor, call `parse_frame(&mut cursor)` to extract the Frame
       - Call `buf.advance(len)` to consume the parsed bytes
       - Return `Ok(Some(frame))`
       - If check returns `Err(ParseError::Incomplete)`, return `Ok(None)`
       - If check returns other error, propagate it

    2. **Check function** `fn check(cursor: &mut Cursor<&[u8]>, config: &ParseConfig, depth: usize) -> Result<(), ParseError>`:
       - If depth > config.max_array_depth, return Invalid error
       - If !cursor.has_remaining(), return Incomplete
       - Read first byte: match on b'+', b'-', b':', b'$', b'*'
       - For +/- : call `find_crlf(cursor)` to verify line is complete
       - For : : call `find_crlf(cursor)`
       - For $ : read decimal length. If -1, return Ok (null). Check len <= config.max_bulk_string_size. Skip len + 2 bytes (data + CRLF)
       - For * : read decimal count. If -1, return Ok (null). Check count <= config.max_array_length. Loop count times calling `check(cursor, config, depth + 1)`
       - Unknown byte: return Invalid with offset

    3. **Parse function** `fn parse_frame(cursor: &mut Cursor<&[u8]>) -> Result<Frame, ParseError>`:
       - Read type byte, match and extract:
       - +: read line bytes, `Frame::SimpleString(Bytes::copy_from_slice(line))`
       - -: read line bytes, `Frame::Error(Bytes::copy_from_slice(line))`
       - :: read line, parse as i64, `Frame::Integer(n)`
       - $: read length. If -1 return `Frame::Null`. Read `len` data bytes as `Bytes::copy_from_slice(data)`, skip trailing CRLF, return `Frame::BulkString(data)`
       - *: read count. If -1 return `Frame::Null`. Loop count times calling `parse_frame` recursively, collect into Vec, return `Frame::Array(vec)`

    4. **Helper functions:**
       - `fn get_u8(cursor: &mut Cursor<&[u8]>) -> Result<u8, ParseError>` -- read one byte or Incomplete
       - `fn find_crlf(cursor: &mut Cursor<&[u8]>) -> Result<&[u8], ParseError>` -- find \r\n, return the line bytes (without CRLF), advance cursor past CRLF. Return Incomplete if no CRLF found.
       - `fn read_decimal(cursor: &mut Cursor<&[u8]>) -> Result<i64, ParseError>` -- read CRLF-terminated decimal integer
       - `fn skip(cursor: &mut Cursor<&[u8]>, n: usize) -> Result<(), ParseError>` -- advance cursor by n or Incomplete

    Write all tests as `#[cfg(test)] mod tests` block inside parse.rs.
  </action>
