Create `src/storage/compact_value.rs` with a 16-byte `CompactValue` struct implementing SSO:

```rust
// Constants
const HEAP_MARKER: u32 = 0xF0000000;   // high nibble = 0xF means heap-allocated
const TYPE_MASK: u32 = 0xF0000000;     // high nibble of len_and_tag
const LEN_MASK: u32 = 0x0FFFFFFF;      // lower 28 bits = length
const SSO_MAX_LEN: usize = 12;

// Inline type tags (high nibble of len_and_tag, for SSO values)
const TAG_STRING: u32 = 0x00000000;    // 0 in high nibble
const TAG_INTEGER: u32 = 0x60000000;   // 6 in high nibble (reserved for future)

// Heap type tags (low 3 bits of pointer)
const HEAP_TAG_STRING: usize = 0;
const HEAP_TAG_HASH: usize = 1;
const HEAP_TAG_LIST: usize = 2;
const HEAP_TAG_SET: usize = 3;
const HEAP_TAG_ZSET: usize = 4;

#[repr(C)]
pub struct CompactValue {
    len_and_tag: u32,
    payload: [u8; 12],
}
const _: () = assert!(std::mem::size_of::<CompactValue>() == 16);
```

**SSO path (values <= 12 bytes):**
- `len_and_tag`: high nibble = type tag (0=string), lower 28 bits = byte length
- `payload[0..len]`: inline data bytes

**Heap path (values > 12 bytes OR collection types):**
- `len_and_tag`: high nibble = 0xF (HEAP_MARKER), lower 28 bits = byte length (for strings) or 0 (for collections)
- `payload[0..4]`: inline prefix (first 4 bytes of string, or zeros for collections)
- `payload[4..12]`: 8-byte packed pointer = `(raw_ptr as usize) | type_tag_in_low_3_bits`

For heap storage, use `Box<RedisValue>` leaked to raw pointer. The type tag in the low 3 bits encodes the RedisValue variant. On access, reconstruct `&RedisValue` by masking off the low 3 bits.

**Implement these methods on CompactValue:**
- `pub fn from_redis_value(value: RedisValue) -> Self` -- main constructor, routes SSO vs heap
- `pub fn inline_string(data: &[u8]) -> Self` -- fast path for small strings
- `pub fn as_redis_value(&self) -> RedisValueRef<'_>` -- returns enum of borrowed views (for reads)
- `pub fn as_bytes(&self) -> Option<&[u8]>` -- fast path for string values
- `pub fn into_redis_value(self) -> RedisValue` -- consuming conversion back (for removal/serialization)
- `pub fn to_redis_value(&self) -> RedisValue` -- cloning conversion (for RDB snapshot clone)
- `pub fn is_inline(&self) -> bool`
- `pub fn type_name(&self) -> &'static str`
- `pub fn type_tag(&self) -> u8` -- 0-7 type discriminant
- `pub fn estimate_memory(&self) -> usize` -- delegates to inner RedisValue for heap, returns len for inline
- `pub fn as_redis_value_mut(&mut self) -> Option<&mut RedisValue>` -- only for heap values (returns None for inline)

Define `RedisValueRef<'a>` enum for zero-copy borrowed access:
```rust
pub enum RedisValueRef<'a> {
    String(&'a [u8]),
    Hash(&'a HashMap<Bytes, Bytes>),
    List(&'a VecDeque<Bytes>),
    Set(&'a HashSet<Bytes>),
    SortedSet {
        members: &'a HashMap<Bytes, f64>,
        scores: &'a BTreeMap<(OrderedFloat<f64>, Bytes), ()>,
    },
}
```

**Implement Drop** for CompactValue: if heap-allocated, reconstruct `Box<RedisValue>` from the raw pointer (mask off low 3 bits) and drop it. If inline, nothing to drop.

**Implement Clone** for CompactValue: if inline, copy bytes directly. If heap, clone the underlying `RedisValue` and create a new heap CompactValue.

**Implement Debug** for CompactValue.

```rust
pub struct CompactEntry {

/// Type alias for backward compatibility during migration

```

4. Rewrite metadata packing for u32 (was u64):
   - `last_access`: bits [31:16] -- 16-bit relative seconds (wraps every ~18 hours, fine for LRU)
   - `version`: bits [15:8] -- 8-bit (wraps at 255, fine for WATCH within a transaction)
   - `access_counter`: bits [7:0] -- 8-bit Morris counter (same as before)

```rust
#[inline]
fn pack_metadata_u32(last_access: u16, version: u8, access_counter: u8) -> u32 {
    ((last_access as u32) << 16) | ((version as u32) << 8) | (access_counter as u32)
}
```

5. Update all accessor methods on CompactEntry:

   - `access_counter() -> u8`: `(self.metadata & 0xFF) as u8`
   - `set_version(v: u32)`: truncate to u8, pack into bits [15:8]
   - `set_last_access(t: u32)`: truncate to u16, pack into bits [31:16]
