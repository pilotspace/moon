ACL-01: SETUSER creates user with rules
- ACL SETUSER alice on >secret ~* +@all → OK
- ACL GETUSER alice returns Array with username="alice", enabled=true in flags

ACL-02: GETUSER/LIST/DELUSER/WHOAMI
- ACL LIST returns Array containing a string starting with "user alice"
- ACL WHOAMI after AUTH alice secret returns "alice"
- ACL DELUSER alice returns Integer 1
- After delete, AUTH alice secret → WRONGPASS

ACL-03: AUTH username password
- AUTH alice secret → OK (when alice exists with >secret)
- AUTH alice wrongpass → WRONGPASS
- AUTH nonexistent pass → WRONGPASS (user not found = WRONGPASS, not ERR)
- Two-argument AUTH on a server started without requirepass

ACL-04: Default user backward compat
- Server started with --requirepass mysecret
- AUTH mysecret (1-arg) → OK
- AUTH default mysecret (2-arg) → OK
- AUTH wrongpass → WRONGPASS
- Non-auth server: no requirepass, any command works without AUTH

ACL-05: NOPERM for denied commands
- Create user bob with +@read -@write (or explicit: +@read +@string -set -mset)
- AUTH bob →OK
- GET somekey → allowed (returns nil if not set, not NOPERM)
- SET somekey value → NOPERM (not ERR)
- NOPERM error starts with "NOPERM " prefix

ACL-06: Key pattern restriction
- Create user restricted with on >pass ~cache:* +@all
- AUTH restricted pass
- SET cache:foo bar → OK
- SET other:key bar → NOPERM (key not matching ~cache:*)
- MSET cache:a 1 other:b 2 → NOPERM (other:b fails check)
- GET cache:foo → OK

ACL-07: ACL LOG
- Create user bob with -@write, attempt SET bob_key val
- ACL LOG (as admin on separate connection) returns Array with entry containing:
  - reason = "command"
  - username = "bob"
  - object containing the command name
- ACL LOG RESET → OK; ACL LOG returns empty array
- ACL LOG 1 after two denied attempts returns 1 entry

ACL-08: SAVE and LOAD
- ACL SETUSER persisted_user on >pass ~* +@all
- ACL SAVE → OK (write to tmpfile)
- ACL LOAD → OK (reload from file)
- After LOAD, ACL GETUSER persisted_user still returns the user
- File content is Redis-compatible: starts with "user persisted_user on"

CAT smoke test:
- ACL CAT returns Array of BulkString names
- ACL CAT @string returns Array containing "get", "set", etc.
