Create `src/command/set.rs`. Add `use rand::seq::IndexedRandom;` for SRANDMEMBER/SPOP.

**Commands to implement:**

1. `sadd(db, args)` -- SADD key member [member ...]. Insert members. Return Integer(count of new members added).

2. `srem(db, args)` -- SREM key member [member ...]. Remove members. Return Integer(count removed). Remove key if set becomes empty.

3. `smembers(db, args)` -- SMEMBERS key. Return Array of all members. Empty array if key missing.

4. `scard(db, args)` -- SCARD key. Return Integer(set.len()) or 0.

5. `sismember(db, args)` -- SISMEMBER key member. Return Integer(1) if member exists, Integer(0) if not.

6. `smismember(db, args)` -- SMISMEMBER key member [member ...]. Return Array of Integer(1) or Integer(0) for each member.

7. `sinter(db, args)` -- SINTER key [key ...]. Collect all sets as cloned HashSets first (to avoid borrow conflicts). Intersect them. Return Array of members. If any key missing, return empty array. WRONGTYPE if any key is wrong type.

8. `sunion(db, args)` -- SUNION key [key ...]. Union of all sets. Missing keys treated as empty sets.

9. `sdiff(db, args)` -- SDIFF key [key ...]. First set minus all others. Missing keys treated as empty sets.

10. `sinterstore(db, args)` -- SINTERSTORE destination key [key ...]. Compute SINTER, store result in destination as new set. Return Integer(result set size).

11. `sunionstore(db, args)` -- SUNIONSTORE destination key [key ...]. Same for SUNION.

12. `sdiffstore(db, args)` -- SDIFFSTORE destination key [key ...]. Same for SDIFF.

13. `srandmember(db, args)` -- SRANDMEMBER key [count]. Without count: return one random BulkString or Null. With positive count: return Array of min(count, set_size) distinct elements using `choose_multiple`. With negative count: return Array of abs(count) elements with possible duplicates using repeated `choose`.

14. `spop(db, args)` -- SPOP key [count]. Like SRANDMEMBER but removes chosen elements. Without count: remove and return one random member. With count: remove and return Array. Remove key if set becomes empty.

15. `sscan(db, args)` -- SSCAN key cursor [MATCH pattern] [COUNT count]. Same cursor logic as HSCAN but iterates set members. Return Array of [next_cursor, Array of members].

**For SRANDMEMBER/SPOP random access:** Convert HashSet to Vec first (collect iter), then use `rand::seq::IndexedRandom::choose` / `choose_multiple` with `&mut rand::rng()`.

**Borrow checker strategy for set algebra:** For SINTER/SUNION/SDIFF, iterate args collecting cloned sets into `Vec<HashSet<Bytes>>` before doing set operations. This releases the borrow on db.

**Tests:**
- sadd/srem: add members, verify count, remove, verify set deletion when empty
- smembers/scard: verify members list and count
- sismember/smismember: membership checks
- sinter: intersection of 2+ sets, empty key means empty result
- sunion: union of sets
- sdiff: difference of sets
- sinterstore/sunionstore/sdiffstore: store results, verify destination
- srandmember: with positive count (distinct), negative count (may duplicate)
- spop: removes elements, set auto-deleted when empty
- sscan: cursor iteration with match
- WRONGTYPE: sadd on string key
