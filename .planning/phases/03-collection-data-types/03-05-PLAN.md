**Internal helper -- CRITICAL for Pitfall 2 (dual structure consistency):**
```rust
fn zadd_member(
    members: &mut HashMap<Bytes, f64>,
    scores: &mut BTreeMap<(OrderedFloat<f64>, Bytes), ()>,
    member: Bytes,
    score: f64,
) -> bool {
    // Remove old entry if exists (MUST remove from both)
    let is_new = if let Some(old_score) = members.remove(&member) {
        scores.remove(&(OrderedFloat(old_score), member.clone()));
        false
    } else {
        true
    };
    members.insert(member.clone(), score);
    scores.insert((OrderedFloat(score), member), ());
    is_new
}

fn zrem_member(
    members: &mut HashMap<Bytes, f64>,
    scores: &mut BTreeMap<(OrderedFloat<f64>, Bytes), ()>,
    member: &[u8],
) -> bool {
    if let Some(score) = members.remove(member) {
        scores.remove(&(OrderedFloat(score), Bytes::copy_from_slice(member)));
        true
    } else {
        false
    }
}
```

**Score boundary parser helper (for Pitfall 6):**
```rust
enum ScoreBound {
    Inclusive(f64),
    Exclusive(f64),
    NegInf,
    PosInf,
}
fn parse_score_bound(s: &[u8]) -> Result<ScoreBound, Frame> {
    let s_str = std::str::from_utf8(s).map_err(|_| err("ERR ..."))?;
    if s_str == "-inf" { return Ok(ScoreBound::NegInf); }
    if s_str == "+inf" || s_str == "inf" { return Ok(ScoreBound::PosInf); }
    if let Some(rest) = s_str.strip_prefix('(') {
        let val: f64 = rest.parse().map_err(|_| err("ERR ..."))?;
        if val.is_nan() { return Err(err("ERR ...")); }
        return Ok(ScoreBound::Exclusive(val));
    }
    let val: f64 = s_str.parse().map_err(|_| err("ERR ..."))?;
    if val.is_nan() { return Err(err("ERR ...")); }
    Ok(ScoreBound::Inclusive(val))
}
```

**Commands:**

1. `zadd(db, args)` -- ZADD key [NX|XX] [GT|LT] [CH] score member [score member ...]. Parse optional flags NX (only add new), XX (only update), GT (only update if new score greater), LT (only update if new score less), CH (return count of changed, not just added). For each score-member pair, parse score as f64, reject NaN. Use zadd_member helper. Return Integer(count).

2. `zrem(db, args)` -- ZREM key member [member ...]. Use zrem_member helper. Return Integer(removed_count). Remove key if sorted set becomes empty.

3. `zscore(db, args)` -- ZSCORE key member. Look up in members HashMap. Return BulkString(score formatted) or Null.

4. `zcard(db, args)` -- ZCARD key. Return Integer(members.len()) or 0.

5. `zincrby(db, args)` -- ZINCRBY key increment member. Parse increment as f64. Get current score (default 0.0), add increment, use zadd_member to update both structures. Return BulkString(new_score_formatted).

6. `zrank(db, args)` -- ZRANK key member. Get member's score from HashMap. Then count elements in BTreeMap that come before (OrderedFloat(score), member) using `.range(..key).count()`. Return Integer(rank) or Null if not found. O(n) per research decision.

7. `zrevrank(db, args)` -- ZREVRANK key member. Same but count elements that come AFTER. rank = total - 1 - forward_rank.

8. `zpopmin(db, args)` -- ZPOPMIN key [count]. Pop entries with lowest scores from BTreeMap (use `first_key_value()` or iterate). Remove from both structures. Default count=1. Return Array of alternating member, score pairs.

9. `zpopmax(db, args)` -- ZPOPMAX key [count]. Same but highest scores (use `last_key_value()` or reverse iterate).

10. `zscan(db, args)` -- ZSCAN key cursor [MATCH pattern] [COUNT count]. Iterate members, return Array of [cursor, Array of alternating member, score_string].

**Add to src/command/sorted_set.rs:**

1. `zrange(db, args)` -- ZRANGE key min max [BYSCORE|BYLEX] [REV] [LIMIT offset count] [WITHSCORES].
   - Default (no BYSCORE/BYLEX): by rank. min/max are integer indices (0-based). Iterate BTreeMap, skip to min, take (max-min+1). REV reverses iteration.
   - BYSCORE: min/max are score bounds (use parse_score_bound). Iterate BTreeMap range. REV reverses.
   - BYLEX: min/max are lex bounds ([ or ( prefix, + and - for inf). All members must have same score for meaningful results. Iterate BTreeMap, filter by member bytes comparison.
   - LIMIT offset count: skip offset results, return up to count.
   - WITHSCORES: include scores in response (alternating member, score).
   - Return Array of members (or member/score pairs with WITHSCORES).

2. `zrevrange(db, args)` -- ZREVRANGE key start stop [WITHSCORES]. Legacy command. Equivalent to ZRANGE with REV by rank. Parse start/stop as indices, iterate BTreeMap in reverse order.

3. `zrangebyscore(db, args)` -- ZRANGEBYSCORE key min max [WITHSCORES] [LIMIT offset count]. Legacy command. Parse min/max as score bounds (per Pitfall 6: handle -inf, +inf, ( prefix). Use BTreeMap range query. Return Array.

4. `zrevrangebyscore(db, args)` -- ZREVRANGEBYSCORE key max min [WITHSCORES] [LIMIT offset count]. Note: args order is max then min (reversed from ZRANGEBYSCORE). Iterate in reverse score order.

5. `zcount(db, args)` -- ZCOUNT key min max. Parse score bounds. Count elements in BTreeMap range. Return Integer.

6. `zlexcount(db, args)` -- ZLEXCOUNT key min max. Parse lex bounds ([ or ( prefix, - and + for inf). Count members in lex range (assuming all same score). Return Integer.

7. `zunionstore(db, args)` -- ZUNIONSTORE destination numkeys key [key ...] [WEIGHTS weight [weight ...]] [AGGREGATE SUM|MIN|MAX]. Collect all sorted sets, compute union with weighted scores aggregated per AGGREGATE policy (default SUM). Store result as new sorted set in destination. Return Integer(result_size).

8. `zinterstore(db, args)` -- ZINTERSTORE destination numkeys key [key ...] [WEIGHTS weight [weight ...]] [AGGREGATE SUM|MIN|MAX]. Same but intersection.
