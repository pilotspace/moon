2. **src/command/string.rs** -- Create all core string command handlers. Each is a pure function. Create a helper `fn err_wrong_args(cmd: &str) -> Frame` that returns `Frame::Error(Bytes::from(format!("ERR wrong number of arguments for '{}' command", cmd)))`. Also create `fn extract_bulk_string(frame: &Frame) -> Option<&Bytes>` helper.

   **GET** (`pub fn get(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 1 arg. Return err_wrong_args("GET") otherwise.
   - Extract key as BulkString bytes. Call `db.get(key)`. If Some(entry) and entry.value is String(v), return `Frame::BulkString(v.clone())`. If None, return `Frame::Null`.

   **SET** (`pub fn set(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: at least 2 args (key, value, optional flags). Return err_wrong_args("SET") if < 2.
   - Parse args[0] as key, args[1] as value (both BulkString).
   - Parse remaining args[2..] in a loop for options:
     - `EX` (next arg is seconds as integer string) -> `expiry = Some(Duration::from_secs(n))`
     - `PX` (next arg is milliseconds as integer string) -> `expiry = Some(Duration::from_millis(n))`
     - `EXAT` (next arg is unix timestamp seconds) -> compute Duration from now
     - `PXAT` (next arg is unix timestamp millis) -> compute Duration from now
     - `NX` -> `nx = true`
     - `XX` -> `xx = true`
     - `KEEPTTL` -> `keepttl = true`
     - `GET` -> `get_old = true`
     - Compare option names case-insensitively (uppercase the arg bytes).
   - If nx and xx both true: if `get_old`, return old value or Null; else return Null.
   - If nx and key exists (db.exists): if `get_old`, return old value; else return Null.
   - If xx and key does not exist: if `get_old`, return Null; else return Null.
   - If `get_old`: save old value before overwriting.
   - If `keepttl`: preserve existing entry's expires_at.
   - Create Entry with value, computed expires_at, Instant::now(). Call db.set(key, entry).
   - Return: if `get_old`, return old value (BulkString) or Null. Else return `Frame::SimpleString(Bytes::from_static(b"OK"))`.
   - For EXAT/PXAT: use `std::time::SystemTime::UNIX_EPOCH + Duration::from_secs(ts)` to get SystemTime, then compute duration from `SystemTime::now()`. Convert that duration to an Instant offset: `Instant::now() + remaining_duration`. If timestamp is in the past, still set the key but with an already-expired instant (lazy expiry will clean it up on next access).

   **MGET** (`pub fn mget(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: at least 1 arg. Return err_wrong_args("MGET") if 0.
   - For each arg (key), call `db.get(key)`. If Some, push BulkString(value). If None, push Null.
   - Return `Frame::Array(results)`. MUST have exactly args.len() elements (Pitfall 7).

   **MSET** (`pub fn mset(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: must be even and >= 2. Return `ERR wrong number of arguments for 'MSET' command` if odd or 0.
   - Iterate pairs (key, value). Call db.set_string(key, value) for each.
   - Return `Frame::SimpleString(Bytes::from_static(b"OK"))`.

   **INCR** (`pub fn incr(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 1. Call `incrby_internal(db, key, 1)`.

   **DECR** (`pub fn decr(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 1. Call `incrby_internal(db, key, -1)`.

   **INCRBY** (`pub fn incrby(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 2 (key, increment). Parse increment as i64. Call `incrby_internal(db, key, delta)`.

   **DECRBY** (`pub fn decrby(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 2 (key, decrement). Parse decrement as i64. Call `incrby_internal(db, key, -delta)`.

   **incrby_internal** (private helper):
   - Get current value from db. If missing, treat as 0.
   - If present, parse stored Bytes as UTF-8 then as i64. If parse fails: return `Frame::Error(Bytes::from_static(b"ERR value is not an integer or out of range"))`.
   - Use `current.checked_add(delta)`. If None (overflow): return `Frame::Error(Bytes::from_static(b"ERR increment or decrement would overflow"))`.
   - Store new value as `Bytes::from(new_val.to_string())` using `db.set_string()`. Preserve existing TTL if key existed.
   - Return `Frame::Integer(new_val)`.
   - IMPORTANT: When preserving TTL for INCR on existing key, read the old entry's `expires_at` before overwriting. Use `db.get()` to get expires_at, then `db.set()` with the same expiry.

   **INCRBYFLOAT** (`pub fn incrbyfloat(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 2 (key, increment). Parse increment as f64.
   - Get current value or default "0". Parse as f64. If NaN or parse fails: return ERR.
   - Add values. If result is NaN or infinite: return `Frame::Error(Bytes::from_static(b"ERR increment would produce NaN or Infinity"))`.
   - Format result: use Rust's `format!("{}", result)` but strip trailing zeros after decimal point. If no decimal point, leave as-is. Specifically: if the string contains '.', remove trailing '0' chars, then remove trailing '.' if present. Example: "3.140000" -> "3.14", "3.000000" -> "3", "3.10" -> "3.1".
   - Store formatted string. Return `Frame::BulkString(Bytes::from(formatted))`.

1. **src/command/string.rs** -- Add remaining string commands:

   **APPEND** (`pub fn append(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 2 (key, value).
   - If key exists and is String(existing): concatenate existing + new_value bytes. Store back. Return `Frame::Integer(new_len as i64)`.
   - If key does not exist: create new entry with value. Return `Frame::Integer(value.len() as i64)`.
   - Preserve existing TTL on append.

   **STRLEN** (`pub fn strlen(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 1.
   - If key exists and is String(v): return `Frame::Integer(v.len() as i64)`.
   - If key does not exist: return `Frame::Integer(0)`.

   **SETNX** (`pub fn setnx(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 2 (key, value).
   - If key exists: return `Frame::Integer(0)`.
   - If key does not exist: set key with value, no expiry. Return `Frame::Integer(1)`.

   **SETEX** (`pub fn setex(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 3 (key, seconds, value). Note: order is key, SECONDS, value (not key, value, seconds).
   - Parse seconds as positive i64. If <= 0: return `ERR invalid expire time in 'SETEX' command`.
   - Set key with value and expiry of Duration::from_secs(seconds). Return OK.

   **PSETEX** (`pub fn psetex(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 3 (key, milliseconds, value). Order: key, MILLIS, value.
   - Parse millis as positive i64. If <= 0: return `ERR invalid expire time in 'PSETEX' command`.
   - Set key with value and expiry of Duration::from_millis(millis). Return OK.

   **GETSET** (`pub fn getset(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 2 (key, value).
   - Get old value (via db.get, clone the Bytes if present, or Null).
   - Set new value with no expiry (GETSET removes TTL). Return old value as BulkString or Null.

   **GETDEL** (`pub fn getdel(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: exactly 1 (key).
   - Remove key (db.remove). If existed and was String(v): return `Frame::BulkString(v)`. If not existed: return Null.

   **GETEX** (`pub fn getex(db: &mut Database, args: &[Frame]) -> Frame`):
   - Arity: 1 or more (key, optional options).
   - Get value. If not exists: return Null.
   - Parse optional args for: EX seconds, PX millis, EXAT timestamp, PXAT timestamp_ms, PERSIST.
   - If PERSIST: remove expiry from entry.
   - If EX/PX/EXAT/PXAT: set new expiry on entry.
   - If no options: just return value (no TTL change).
   - Return value as BulkString.
