     ```rust
     const WRITE_COMMANDS: &[&[u8]] = &[
         b"SET", b"MSET", b"SETNX", b"SETEX", b"PSETEX",
         b"GETSET", b"GETDEL", b"GETEX",
         b"INCR", b"DECR", b"INCRBY", b"DECRBY", b"INCRBYFLOAT",
         b"APPEND", b"DEL", b"UNLINK",
         b"EXPIRE", b"PEXPIRE", b"PERSIST",
         b"RENAME", b"RENAMENX",
         b"HSET", b"HMSET", b"HDEL", b"HSETNX", b"HINCRBY", b"HINCRBYFLOAT",
         b"LPUSH", b"RPUSH", b"LPOP", b"RPOP", b"LSET", b"LINSERT", b"LREM", b"LTRIM",
         b"SADD", b"SREM", b"SPOP",
         b"SINTERSTORE", b"SUNIONSTORE", b"SDIFFSTORE",
         b"ZADD", b"ZREM", b"ZINCRBY", b"ZPOPMIN", b"ZPOPMAX",
         b"ZUNIONSTORE", b"ZINTERSTORE",
         b"SELECT",  // Log SELECT so replay tracks db index
     ];
     pub fn is_write_command(name: &[u8]) -> bool {
         WRITE_COMMANDS.iter().any(|c| c.eq_ignore_ascii_case(name))
     }
     ```

   - `pub async fn aof_writer_task(mut rx: mpsc::Receiver<AofMessage>, aof_path: PathBuf, fsync: FsyncPolicy, cancel: CancellationToken)`:
     - Open file in append mode (create if not exists)
     - Use `tokio::io::BufWriter` wrapping `tokio::fs::File`
     - Track last_fsync: Instant for EverySec policy
     - Loop: select on rx.recv() and cancel.cancelled()
     - On Append(data): write_all(&data), then fsync based on policy
     - On Rewrite(db): call `rewrite_aof(db, &aof_path).await` (flushes current writer, generates new AOF, reopens file)
     - On Shutdown or cancel: flush and break
     - For EverySec: also use tokio::time::interval(1s) in the select to fsync periodically

   - `pub fn replay_aof(databases: &mut [Database], path: &Path) -> anyhow::Result<usize>`:
     - Read entire file into BytesMut
     - Parse frames using `protocol::parse::parse()` in a loop
     - Track selected_db (start at 0)
     - For each frame, call `command::dispatch()` (reuse exact dispatch path per CONTEXT.md)
     - Count commands replayed, return count
     - On parse error: log warning with tracing, return what was loaded so far (partial replay)

   - `pub fn serialize_command(frame: &Frame) -> Bytes`:
     - Use protocol::serialize to convert Frame to RESP bytes
     - Return as frozen Bytes
