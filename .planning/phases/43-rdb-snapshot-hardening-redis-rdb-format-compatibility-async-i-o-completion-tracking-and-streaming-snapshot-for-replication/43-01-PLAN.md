**Constants:**
```rust
const REDIS_RDB_MAGIC: &[u8] = b"REDIS";
const REDIS_RDB_VERSION: &[u8] = b"0010";
const RDB_OPCODE_AUX: u8 = 0xFA;
const RDB_OPCODE_RESIZEDB: u8 = 0xFB;
const RDB_OPCODE_EXPIRETIME_MS: u8 = 0xFC;
const RDB_OPCODE_SELECTDB: u8 = 0xFE;
const RDB_OPCODE_EOF: u8 = 0xFF;
const RDB_TYPE_STRING: u8 = 0;
const RDB_TYPE_LIST: u8 = 1;
const RDB_TYPE_SET: u8 = 2;
const RDB_TYPE_ZSET_2: u8 = 5;
const RDB_TYPE_HASH: u8 = 4;
```

**Helper functions:**
- `fn write_length(buf: &mut Vec<u8>, len: u64)` - Redis variable-length encoding: 6-bit (00xxxxxx), 14-bit (01xxxxxx + byte), 32-bit (10000000 + 4 BE bytes), 64-bit (10000001 + 8 BE bytes)
- `fn write_redis_string(buf: &mut Vec<u8>, s: &[u8])` - length-prefixed string
- `fn write_aux(buf: &mut Vec<u8>, key: &[u8], value: &[u8])` - 0xFA + key string + value string
- `fn write_rdb_header(buf: &mut Vec<u8>)` - magic + version + AUX fields (redis-ver=7.0.0, redis-bits=64, ctime=now, used-mem=0)
- `fn write_rdb_footer(buf: &mut Vec<u8>)` - 0xFF + CRC64 of entire buf as 8-byte LE

**Entry writer:**
- `fn write_rdb_entry(buf: &mut Vec<u8>, key: &[u8], entry: &Entry, base_ts: u32)` - match on `entry.value.as_redis_value()`:
  - String: optional 0xFC expiry + type 0 + key string + value string
  - Hash/HashListpack: type 4 + key + length(fields) + for each field: field_string + value_string
  - List/ListListpack: type 1 + key + length(elements) + for each: element_string
  - Set/SetListpack/SetIntset: type 2 + key + length(members) + for each: member_string
  - SortedSet/SortedSetBPTree/SortedSetListpack: type 5 + key + length(members) + for each: member_string + f64 LE bytes (8 bytes)
  - Stream: serialize as string (JSON or custom) with type 0 for simplicity (streams in Redis RDB use complex type 19+; for PSYNC2 to our replicas we don't need Redis-native stream encoding)

**Public API:**
- `pub fn write_rdb(databases: &[Database], buf: &mut Vec<u8>)` - header + for each non-empty db: SELECTDB + RESIZEDB(db_size, expires_size) + entries + footer
- `pub fn save(databases: &[Database], path: &Path) -> anyhow::Result<()>` - write_rdb to Vec, atomic write (tmp + rename)
- `pub fn load_rdb(databases: &mut [Database], data: &[u8]) -> anyhow::Result<usize>` - verify magic/version, verify CRC64, parse SELECTDB/RESIZEDB/entries/EOF, return key count

**Reading helpers (for load_rdb):**
- `fn read_length(cursor: &mut Cursor<&[u8]>) -> anyhow::Result<(u64, bool)>` - returns (value, is_encoded) where is_encoded=true for special integer encodings (11xxxxxx prefix)
- `fn read_redis_string(cursor: &mut Cursor<&[u8]>) -> anyhow::Result<Vec<u8>>` - handle length-prefixed and integer-encoded strings (0xC0=8-bit int, 0xC1=16-bit LE int, 0xC2=32-bit LE int)
